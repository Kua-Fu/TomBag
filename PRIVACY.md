# TomBag Privacy Policy

Last updated: 2026-05-22

TomBag is a local developer toolkit for JSON viewing, timestamp conversion, and unit conversion. This policy explains how the TomBag web app and Chrome extension handle data.

## Data Collection

TomBag does not collect, sell, rent, or share personal information.

TomBag does not use analytics, tracking pixels, advertising identifiers, or remote telemetry.

## Local Processing

All tool inputs are processed locally in your browser. This includes JSON text, imported files, timestamps, unit values, copied output, and conversion results.

TomBag may store the following preferences locally in your browser:

- Language preference.
- Background on/off preference.
- Optional custom background image selected by you.

These preferences stay on your device and are used only to restore your chosen interface settings.

## Data Sharing

TomBag does not transmit your tool inputs or local preferences to the developer, third-party services, or external servers.

## Permissions

The Chrome extension may request browser storage permission so it can keep local interface settings. This permission is not used to collect or transmit personal data.

## Changes

If this privacy policy changes, the updated version will be published in this repository.

## Contact

For questions about this privacy policy, please open an issue in the TomBag GitHub repository.

# TomBag 隐私政策

更新日期：2026-05-22

TomBag 是一个本地开发小工具集，提供 JSON 展示、时间戳转换和单位换算能力。本政策说明 TomBag 网页应用和 Chrome 扩展如何处理数据。

## 数据收集

TomBag 不收集、出售、出租或分享个人信息。

TomBag 不使用分析统计、跟踪像素、广告标识符或远程遥测。

## 本地处理

所有工具输入都会在你的浏览器本地处理，包括 JSON 文本、导入文件、时间戳、单位数值、复制内容和换算结果。

TomBag 可能会在浏览器本地保存以下偏好设置：

- 语言偏好。
- 背景开启或关闭偏好。
- 你主动选择的自定义背景图片。

这些偏好只保留在你的设备上，仅用于恢复你选择的界面设置。

## 数据共享

TomBag 不会将你的工具输入或本地偏好传输给开发者、第三方服务或外部服务器。

## 权限

Chrome 扩展可能会请求浏览器存储权限，用于保存本地界面设置。该权限不会用于收集或传输个人数据。

## 政策变更

如果本隐私政策发生变化，更新后的版本会发布在本仓库中。

## 联系方式

如对本隐私政策有疑问，请在 TomBag GitHub 仓库中提交 issue。
